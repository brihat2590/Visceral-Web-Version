# Extractable Components

## High Value Layout Components
- `src/components/LandingNav.tsx` -> reusable landing navigation
- `src/components/HeroContent.tsx` -> reusable hero + CTA + preview block
- `src/components/HowItWorks.tsx` -> reusable tabbed process section
- `src/components/FeaturesSection.tsx` -> reusable value proposition section
- `src/components/Footer.tsx` -> reusable site footer

## Lower Priority / App Shell Components
- `src/components/VisceralSidebar.tsx` -> dashboard shell sidebar

## Notes
- No dedicated button, card, or input primitives are required for the landing page draft.
- The landing page currently relies mostly on direct Tailwind markup inside these section components.
